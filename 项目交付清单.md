## 🚀 启动步骤

### 1. 数据库初始化

**Windows:**
```bash
init-database.bat
```

**Linux/Mac:**
```bash
chmod +x init-database.sh
./init-database.sh
```

### 2. 启动后端

```bash
cd game-guide-backend
mvn spring-boot:run
```

后端地址: http://localhost:8080/api

### 3. 启动前端

```bash
cd frontend
npm install
npm run dev
```

前端地址: http://localhost:5173

### 4. 登录测试

使用 `admin` / `admin123` 登录系统

## 📝 API 接口总结

### 认证接口（无需 Token）
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/login` - 用户登录

### 业务接口（需要 Token）
- `GET /api/games` - 游戏列表（所有用户）
- `POST /api/games` - 创建游戏（仅 ADMIN）
- `PUT /api/games/{id}` - 更新游戏（仅 ADMIN）
- `DELETE /api/games/{id}` - 删除游戏（仅 ADMIN）

其他接口类似，详见运行指南。
