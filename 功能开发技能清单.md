# 🎯 游戏攻略知识库系统 - 功能开发技能清单

## 📚 技能分类

### 一、后端开发技能

#### 1. 用户认证与授权 (Security & JWT)

**核心技能：**
- ✅ Spring Security 6.x 配置与集成
- ✅ JWT Token 生成、验证与刷新机制
- ✅ BCrypt 密码加密算法应用
- ✅ 自定义 JWT 过滤器开发
- ✅ UserDetailsService 接口实现
- ✅ 认证异常处理与统一响应

**涉及文件：**
- `SecurityConfig.java` - Spring Security 核心配置
- `JwtUtil.java` - JWT 工具类封装
- `JwtAuthenticationFilter.java` - 请求拦截与 Token 验证
- `CustomUserDetailsService.java` - 用户加载服务
- `JwtAuthenticationEntryPoint.java` - 认证失败处理
- `AuthController.java` - 登录注册接口
- `AuthServiceImpl.java` - 认证业务逻辑

**技术要点：**
- 无状态会话管理 (SessionCreationPolicy.STATELESS)
- 请求拦截器链配置
- Token 解析与用户信息提取
- 密码编码器配置
- 认证管理器配置

---

#### 2. 权限控制 (RBAC)

**核心技能：**
- ✅ 基于角色的访问控制 (Role-Based Access Control)
- ✅ 方法级权限注解 (@PreAuthorize, @Secured)
- ✅ URL 级权限配置
- ✅ 动态权限判断
- ✅ 权限异常处理

**涉及文件：**
- `SecurityConfig.java` - 权限规则配置
- `User.java` - 角色字段定义
- 各 Controller - 权限注解应用

**技术要点：**
- requestMatchers() 配置不同 HTTP 方法的权限
- hasRole() 角色判断
- permitAll() 公开接口配置
- authenticated() 认证要求
